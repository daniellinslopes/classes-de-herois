## Sobre o Desafio

Este é o **desafio final** do curso **Blip - Lógica de Programação** da DIO (Digital Innovation One), parte do Bootcamp Blip.  

O objetivo era consolidar os principais conceitos aprendidos ao longo da trilha:  
- Variáveis  
- Operadores  
- Laços de repetição  
- Estruturas de decisão (if/else, switch)  
- Funções / Métodos  
- **Classes e Objetos** (foco principal)

### Requisitos do Projeto

Criar uma **classe genérica** que represente um herói de aventura com as propriedades:  
- `nome`  
- `idade`  
- `tipo` (exemplos: guerreiro, mago, monge, ninja)

A classe deve possuir um método `atacar()` que exibe a mensagem:  
**"o {tipo} atacou usando {ataque}"**

O tipo de ataque varia conforme a classe do herói:

| Tipo       | Ataque usado          |
|------------|-----------------------|
| mago       | magia                 |
| guerreiro  | espada                |
| monge      | artes marciais        |
| ninja      | shuriken              |

## Tecnologias Utilizadas

- JavaScript (ES6+)  
- Classes e Objetos  
- Template Strings  
- Switch Case  
- Laços de repetição (for)  
- Console.log para exibição
